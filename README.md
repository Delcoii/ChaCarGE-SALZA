## Applications
### Vehicle Platform Construction
- Vehicle platform development
- Implementation of longitudinal/lateral control systems

### Traffic Light Recognition Algorithm
- Development of traffic light recognition algorithm using YOLOv11
- ONNX model optimization
- Quantitative evaluation of device-specific performance

### Driving Habit Improvement Algorithm Development
- Definition of good driving habits
- Generation of virtual sensor data
- Development of driving score calculation algorithm based on sensor data

### Communication Environment Construction
- Construction of CAN communication system for each module
- Monitoring system development
- Parsing of various sensor data

### Infotainment GUI Development
- Construction of Yocto Linux-based environment
- Development of infotainment display
